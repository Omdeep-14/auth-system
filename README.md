A comprehensive, security-first authentication system built with the MERN stack, featuring JWT-based authentication, advanced session management, and multiple layers of security protection.

1)Authentication
JWT-based Authentication - Secure token-based auth with access and refresh tokens
Email Verification - Link-based email verification for new accounts
OTP Login - One-Time Password authentication via email
Forgot Password Flow - Secure password reset mechanism (in progress)
Session Management - Single-session enforcement with automatic invalidation
Role-Based Access Control (RBAC) - Protected routes for normal users and users with admin permissions

2)Security
CSRF Protection - Cross-Site Request Forgery prevention using csrf token
XSS Protection -XSS prevention using zod validation
NoSQL Injection Prevention - Database query sanitization
Rate Limiting - Redis-powered request throttling to prevent abuse
